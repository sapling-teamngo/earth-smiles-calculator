Technical Documentation

Introduction

This documentation provides the technical basis for the Earth Smiles Calculator, which designs semicircular bunds for water harvesting based on WOCAT (World Overview of Conservation Approaches and Technologies) methodologies.

WOCAT Methodology Basis

The design parameters are primarily derived from the WOCAT Benin case study "The Half-Moon Technique" which documents successful implementation in West Africa.

Design Equations

1. Structure Diameter Calculation

D = Dbase √ó Sf √ó Rf √ó Soilf √ó Landf

Where:

  • Dbase = 4.0 m (Base diameter from WOCAT Benin)
  • Sf = Slope adjustment factor (0.6-1.0)
  • Rf = Rainfall adjustment factor (rainfall/1025)
  • Soilf = Soil type factor (0.9-1.1)
  • Landf = Land use factor (0.9-1.2)
Source: Adapted from WOCAT Benin Case Study, Pages 1-2

Adjustment Factors

Slope Adjustment Factors

Slope Range (%) Adjustment Factor Rationale
0-2% (Flat) 1.0 Standard conditions
2-5% (Gentle) 0.95 Minor reduction for increased runoff
5-10% (Moderate) 0.9 Moderate reduction
10-15% (Rolling) 0.85 Significant reduction needed
15-25% (Hilly) 0.75 Major reduction for steep slopes
>25% (Steep) 0.6 Maximum reduction

Rainfall Adjustment Factors

Rainfall factor = min(rainfall / 1025, 1.2)

Where 1025mm is the reference rainfall from WOCAT Benin case study.

Implementation Protocol

Based on WOCAT Benin establishment activities (Page 2):

  1. Demarcation: Use compass with 2-meter radius to create semicircle
  2. Excavation: Dig to depth of 15-20 cm using pick, pickaxe, and shovel
  3. Ridge Construction: Build ridges from inside outward, height up to 40 cm
  4. Layout: Arrange in staggered pattern following contour lines
  5. Spacing: Maintain 4-meter spacing between rows
  6. Enrichment: Add 35 kg of compost or manure per structure
  7. Mulching: Apply mulch to minimize moisture loss
